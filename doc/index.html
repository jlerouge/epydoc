Epydoc

Epydoc is a tool for generating API documentation for Python modules, based on their docstrings. See the epydoc API documentation for an example of epydoc's output. A lightweight markup language called epytext can be used to format docstrings, and to add information about specific fields, such as parameters and instance variables.
