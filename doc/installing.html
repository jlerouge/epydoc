Installing Epydoc

Source Install 

[user ~]$ wget http://epydoc.sf.net/epydoc-1.0.tgz
[user ~]$ tar -xvzf epydoc-1.0.tgz
[user ~]$ cd epydoc/src
[user src]$ su
Password:
[root src]# python setup.py install
running install
running build
[...]
copying build/scripts/epydoc -> /usr/bin
changing mode of /usr/bin/epydoc to 100775
[root src]#

If you like, you can delete the source tar file: 

[root src]# cd ../..
[root src]# rm -rf epydoc-1.0
[root src]# rm -rf epydoc-1.0.tgz

Windows Install

  • Download and run epydoc-1.0.win32.exe.
  • Follow the on-screen instructions.
  • (Optional) Delete epydoc-1.0.win32.exe.
  • Create a shortcut for the gui?
